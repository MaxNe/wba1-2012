Vor- und Nachteile

Vorteile

  • In der Regel Kostenlos
  • Einfach einzubinden
  • Weniger Schreibarbeit
  • Fertige Widgets und Animationen
  • Code frei verfügbar
  • Code anpassbar

Vor- und Nachteile

Nachteile

  • Frameworks haben unterschiedliche Syntax
  • Inkompatibilitäten bei paralleler Nutzung von JS-Frameworks

Übung: Einbindung von Javascript Frameworks

  • Downloading jQuery: DOWNLOAD 
     <head> <script src="jquery-1.8.3.min.js"></script> </head>
  • Warum nicht "type" dazu?
    <script type="text/javascript">
    HTML 5 braucht es nicht! JavaScript wird als ursprüngliche Script-Sprache in HTML 5 benutzt.
  • Alternative zum downloading: 
     <head> <script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script> </head>
