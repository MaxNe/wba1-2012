2 Aufbau von HTML Seiten

  • Textdokument mit Endung .html oder .htm
  • Tags

  • Attribute 
    <tagname attributename="wert">

5 Inline und Block Elemente

Block Elemente

  • Erzeugen eine neue Zeile
  • Können Text, Inline- & weitere Block Elemente beinhalten
  • Beispiele: h1-h6, p, div

Inline Elemente

  • Erzeugen keine neue Zeile und sind nebeneinander angeordnet
  • Können nur Text und weitere Inline-Elemente beinhalten
  • Beispiele: a, strong, em, span
