1 Was ist HTML?

  • Hyper Text Markup Language
  • Markup Language
  • keine Programmiersprache
  • Historie
    • 1990
    • Ursprünglich: Wissenschaftliche Dokumente online darstellen

2 Aufbau von HTML Seiten

  • Textdokument mit Endung .html oder .htm
  • Tag 
    <tagname> </tagname>
  • Attribut 
    <tagname attributename="wert">
  • HTML Element 
    <tagname> text </tagname>

3 Darstellung von HTML-Seiten

<!DOCTYPE html>
<html>
	<head>
		<title>Hello World!</title>
	</head>
	<body>
		Hello World!
		Meine erste Webseite.
		Mein Profil
	</body>
</html>

3 Darstellung von HTML-Seiten

<!DOCTYPE html>
<html>
	<head>
		<title>Hello World!</title>
	</head>
	<body>
		<h1>Hello World!</h1>
		<p>Meine erste Webseite.</p>
		<a href="profil.html">Mein Profil</a>
	</body>
</html>
