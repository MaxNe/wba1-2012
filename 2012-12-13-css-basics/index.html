Einbindungsmöglichkeiten

  • Element
  • HTML (Head-Bereich)
  • externe Datei (link / @import)

Element


<p style="color: #FF0000;">
    Das ist ein Absatz in roter Farbe.
</p>

HTML (Head-Bereich)


<head>
    <style>
        <!--
            p {
                color: #FF0000;
            }
        -->
    </style>
</head>

externe Datei


<head>
    <link rel="stylesheet" type="text/css" href="../css/reveal.css">
</head>

oder 


<head>
    <style>
        <!--
            @import "../css/reveal.css";
            @import url("../css/theme/reveal.css");
            @import url("../css/print/pdf.css") print;
        -->
    </style>
</head>

Selektoren

  • Element
  • Klasse (.)
  • ID (#)
  • Kind (>)

Element

HTML 

<body>
    <h1>Element</h1>
</body>
CSS 

h1 {
    color: #77CC00;
}

Klasse

HTML 

<body>
    <h1 class="ueberschrift">Element</h1>
</body>
CSS 

.ueberschrift {
    color: #77CC00;
}

ID

HTML 

<body>
    <h1 id="ueberschrift">Element</h1>
</body>
CSS 

#ueberschrift {
    color: #77CC00;
}

Kind

HTML 

<body>
    <div>
        <h1>Element1</h1>
        <u1>
            <li><h1>Element Liste</h1></li>
        </u1>
        <h1>Element2</h1>
    </div>
</body>
CSS 

div > h1 {
    color: #77CC00;
}
