Attribute des Videotags

autoplay
Startet die Wiedergabe automatisch 
controls
Zeigt Steuerelemente (Play/Pause usw.) an 
loop
Spielt das Video in Schleife ab

Attribute des Videotags

muted
Schaltet das Video stumm 
poster="standbild.jpg"
Zeigt ein Bild, während das Video nicht läuft 
preload="auto|metadata|none"
Gibt an ob und was im Hintergrund geladen werden soll

Steuerelemente mit Javascript anpassen

								    <script type="text/javascript">
      function videoplay() {
        var video = document.getElementById("video");
        var button = document.getElementById("play");
        if (video.paused) {
          video.play();
          button.textContent = "Pause";
        } else {
          video.pause();
          button.textContent = "Play";
        }
      }
      function zumanfang() {
        var video = document.getElementById("video");
        video.currentTime = 0;
      }
      function spulen(wert) {
        var video = document.getElementById("video");
        video.currentTime += wert;
      }      
    </script>

Attribute

currentTimeaktuelle Abspielpositionkann verändert werden
pausedgibt true zurück, wenn das Video pausiert istread only
endedgibt true zurück, wenn das Video zu ende istread only
autoplaytrue wenn das Video automatisch abgespielt werden sollkann verändert werden
