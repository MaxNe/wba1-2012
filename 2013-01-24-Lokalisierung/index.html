Auch alle anderen Zeichen lassen sich so bzw. ähnlich darstellen.

      • 110.182 Darstellbare Zeichen

Schrift

ASCII - American Standard Code for Information Interchange

(ISO 646)

7-Bit-Zeichencodierung (max. 128Zeichen)
Grundlage für alle auf Bits basierende Kodierung für Zeichensätze

  • enthält:
    • Zahlen
    • Lateinisches Alphabet(groß- und kleinbuchstaben)
    • "standard" Zeichen

Schrift

Unicode(ISO 10646)

  • Ziel alle Sprachen und Zeichen abzudecken
  • 17 Ebenen zu je 2^16(65.536 Zeichen)
    (Insgesamt: 1.114.112 mögliche Zeichen)
  • Oktober 1991 - Version 1.0.0 - 7.161 Zeichen
  • September 2012 - Version 6.2 - 110.182 Zeichen

Schrift

UTF-8
Unicode Transformation Format

Standard Internetkodierung für Unicode
nach "Internet Engineering Task Force"
