1. Was ist JavaScript?

  • Skriptsprache
  • keine Datenbanken, Klassen, Schnittstellen und Sicherheitsstufen
  • Variablen ohne Datentypdeklaration
  • steuert das Verhalten einer Webanwendung

2. Welche Einsatzmöglichkeiten gibt es?

  • aufklappende Menüs
  • Dialogboxen zur Eingabe von Daten
  • User-Aktionen können mit Funktionen belegt werden (Events)
  • Pop-Up Fenster
  • Formularüberprüfung auf korrekte Eingabe
  • visuelle Effekte

3. Wie ist JavaScript aufgebaut?

  • Variablen und Werte
  • Objekte, Eigenschaften und Methoden
  • Funktionen
  • Operatoren
  • Bedingte Anweisungen (if-else/switch)
  • Schleifen (while/for/do-while)
Anwendungsbeispiel 

function pruefe(wort){
	if(wort.length<3 || wort.length==0){
		alert("Ungültige Eingabe!");
		document.adresse.stadt.focus();
	}
	else if(wort.length>=3)
		alert("Wow");
}
					

<form name="adresse">
	<input type="text" size="30" name="stadt">
	<input type="button" value="fertsch"
			onclick="pruefe(document.adresse.stadt.value)">
</form>

4. Wie lässt sich JavaScript einbinden?

1. Möglichkeit

Entweder im head-, oder body-Bereich des HTML-Codes 


<script type="text/javascript">
		  //Es wird ein Warnhinweis ausgeworfen
		  alert("Hey Leute, alles klar?");
</script>

2. Möglichkeit

Als externe Datei im head-Bereich einfügen 


<head>
	  <script type="text/javascript" src="/meinscript.js"></script>
</head>

3. Möglichkeit

In Eventhandlern 


<input type="button" value="Klick Mich" onclick="alert('Ich bin ein Event')">
					

<body onload="document.anmeldung.vorname.focus()">

6. Was ist DOM?

  • Document Object Model
  • Beschreibt wie auf Elemente von HTML und anderen XML basierten Dokumenten zugegriffen werden kann
  • Elemente eines HTML-Dokuments werden durch Baumstruktur in Beziehung gesetzt
  • Wird allmählich von allen gängigen Browsern verwendet
