Was ist eine API?

  • Application Programming Interface
  • Schnittstelle zur Anwendungsprogrammierung
  • Webseite --> Graph API --> Facebook
  • Graph API
    • erlaubt Schreib- und Lesezugriff auf die meisten Objekte und Verbindungen, die den sozialen Graphen von Facebook ausmachen

Sozialer Graph

  • Speicherung der Objekte
  • jedes Objekt besitzt Identifikationsnummer
  • Basis URL: https://graph.facebook.com
  • Besipiel URL: https://graph.facebook.com/ObjektID

  • https://graph.facebook.com/100005051133031
  • https://graph.facebook.com/tomoko.alekandero

JSON

  • JavaScript Object Notation

  • unabhängig von der Programmiersprache
