Was sind die Chrome Developer Tools?

  • fester Bestandteil des Chrome Webbrowsers
  • basieren auf dem WebKit Web Inspector
  • ermöglichen es HTML & CSS Code...
    • ...einzusehen
    • ...selbigen zu manipulieren
    • ...zu debuggen
  • Änderungen sind "on the fly" sichtbar aber nicht persistent

Wie kann ich sie benutzen?

    Windows / MAC OSX (im Chrome)
  • Control + Shift + I - öffnet Developer Tools
  • Control + Shift + C - aktiviert Inspect Element Modus
  • Command + Option + I - öffnet Developer Tools
  • Control + Option + C - aktiviert Inspect Element Modus
