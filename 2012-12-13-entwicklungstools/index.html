Was sind die Chrome Developer Tools?

  • fester Bestandteil des Chrome Webbrowsers
  • basieren auf dem WebKit Web Inspector
  • ermöglichen es HTML & CSS Code...
    • ...einzusehen
    • ...selbigen zu manipulieren
    • ...zu debuggen
  • Änderungen sind "on the fly" sichtbar aber nicht persistent

Wie kann ich sie benutzen?

    Windows / MAC OSX (im Chrome)
  • Control + Shift + I - öffnet Developer Tools
  • Command + Option + I - öffnet Developer Tools
  • Control + Shift + C - aktiviert Inspect Element Modus
  • Control + Option + C - aktiviert Inspect Element Modus

HTML Tools

HTML Tools

  • zeigen DOM in Baumform
  • HTML Code editierbar durch Doppelklick
  • ermöglichen Navigation und Bearbeitung des HTML Dokumentes
  • Markierung ausgewählter Knoten innerhalb der Seite inkl. Informationen (z.B. Bildgröße etc.)

CSS Tools

CSS Tools

  • CSS Regeln deaktivierbar
  • verschiedene Komfortfunktionen wie Colorpicker
  • zeigen relevante CSS Regeln an, sowie deren Vererbungen etc.
  • computed style zeigt die im Browser angewandten Regeln für jedes Element an
  • CSS Code vollständig editierbar, sogar für verschiedene Status (active, hover etc.)

Live Editing and Revision History

  • Der Editor im Browser...naja fast.
  • ermöglichst direktes Bearbeiten der Stylesheets inkl. Live Änderung
  • abspeichern der Stylesheets lokal möglich
  • Revision History zeigt Änderungen übersichtlich dar und ermöglicht es, Änderungen rückgängig zu machen
