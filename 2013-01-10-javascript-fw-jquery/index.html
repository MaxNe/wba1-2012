Hinweis: Um Versionsübergreifend gültig zu sein, kann man anstatt 1.8.3 auch nur 1 schreiben. Dann wird die aktuellste Version zwischen 1.0.0 und 1.9.9 eingebunden.

Let's start writing less

Syntax (1)

$(selector).action()

$ := jQuery-Notation
selector := CSS-Selector
action := jQuery-Funktion 

$("p").hide() - Blendet alle <p> Elemente aus.

$("#test","p").hide() - Blendet Elemente mit der id "test" und alle <p> Elemente aus.

Get und Set (1)

  • text() - Setzt oder gibt den Text von Elementen wieder
  • html() - Setzt oder gibt den Inhalt von Elementen in HTML-Struktur wieder
  • val() - Setzt oder gibt den Inhalt von Formularfeldern wieder

Basisfunktionen

Add und Remove

  • append() - Fügt Inhalt am Ende des ausgewählten Elements ein
  • prepend()- Fügt Inhalt am Anfang des ausgewählten Elements ein
  • before() - Fügt Inhalt vor dem ausgewähltem Element ein
  • after() - Fügt Inhalt hinter dem ausgewähltem Element ein
  • remove() - Löscht das ganze Element und die Kind-Elemente
  • empty() - Löscht nur die Kind-Elemente aus dem ausgewähltem Element

Übung 3

Fotoslideshow mit roundabout.js und reflection.js

Um zu zeigen, wie mächtig jQuery Plugins sind, sollst du hier das Plugin roundabout einsetzen. Es macht aus einer einfachen unordered list eine Coverflow-Bibliothek. Mit dem Plugin reflection kann man zudem noch eine Spiegelung auf den Boden realisieren.
Es sind sehr einfache Beispiele, die sich schnell umsetzen lassen um den Rahmen der Übung nicht zu sprengen. Prinzipiell lässt sich aber jedes Plugin derart leicht einbinden und anwenden.
