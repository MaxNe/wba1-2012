Syntax (1)


$(selector).action()

$ := jQuery-Notation; selector := CSS-Selector; action := jQuery-Funktion 

$("p").hide() - Blendet alle <p> Elemente aus.

$("#test","p").hide() - Blendet alle Elemenete mit der id "test" und alle <p> Elemente aus.

Add und Remove

  • append() - Fügt Inhalt am Ende des ausgewählten Elements ein
  • prepend()- Fügt Inhalt am Anfang des ausgewählten Elements ein
  • before() - Fügt Inhalt vor dem ausgewähltem Element ein
  • after() - Fügt Inhalt hinter dem ausgewähltem Element ein
  • remove() - Löscht das ganze Element und die Kindselemente
  • empty() - Löscht nur die Kindselemente aus dem ausgewähltem Element
